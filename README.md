# Reliance Smart Sales Report - Power BI

## Overview
This project is a Power BI sales report for Reliance Smart. It is designed to provide insights into sales performance, customer behavior, and regional trends using visual dashboards and data analysis.

## Files in Repository
- **Calendar.csv** - Contains date-related data to help manage timelines and date-based insights.
- **Customers.csv** - Customer data, including demographics and customer segmentation details.
- **Products.csv** - Product catalog information for sales tracking.
- **Regions.csv** - Geographic data for analyzing regional performance.
- **Reliance Smart Sales Report.pbix** - Power BI report file containing the visual dashboard and data model.
- **Reliance Smart logo.png** - Company logo for branding within the Power BI report.
- **Returns-1997-1998.csv** - Data on product returns from the years 1997-1998.
- **Stores.csv** - Store location and relevant details for retail analysis.

## Power BI Report Features
- **Sales Analysis:** Detailed insights into sales trends over time, product performance, and customer preferences.
- **Regional Insights:** Interactive maps to visualize sales distribution across various regions.
- **Customer Segmentation:** Insights on customer demographics and purchasing behaviors.
- **Returns Analysis:** Analysis of product returns to identify potential issues in quality or customer satisfaction.

## How to Use
1. Download the repository files.
2. Open `Reliance Smart Sales Report.pbix` in Power BI Desktop.
3. Ensure all data files are properly linked in the Power BI data model.
4. Explore the visual dashboards and customize as needed.

## Prerequisites
- Power BI Desktop (latest version recommended)
- Basic understanding of Power BI report navigation
